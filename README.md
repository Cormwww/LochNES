# LochNES
A (soon to be) fully-featured NES emulator written in Rust. 

## Roadmap
- [X] CPU
- [ ] Bus
- [ ] PPU
- [ ] APU

## Illegal Opcodes
At the moment, the emulator does not implement any illegal opcodes. If it encounters one, it will panic.
