# LochNES
A (soon to be) fully-featured NES emulator written in Rust. 

## Roadmap

### Addressing modes
- [x] Implicit
- [x] Accumulator
- [x] Immediate
- [x] Zero Page
- [x] Zero Page, X
- [x] Zero Page, Y
- [x] Absolute
- [x] Absolute, X
- [x] Absolute, Y
- [x] Indirect
- [x] Indexed Indirect
- [x] Indirect Indexed
